Classes in Ruby

Essentially everything in ruby is a object or class, anything can be called upon to see what class they belong to.
Examples:

p 3.class => Fixnum
p "Hello World".class => String
p [1,2,3].class => Array
p hash = {'a'=>1,'b'=>2}.class => Hash

So basically all the methods we are using for objects in ruby and defined in the classes of those objects.
A simple example would be the capitalize method in the String Class.
Now we know its a Class so basically this would be the syntax for it

class String
def capitalize
self[0].upcase + self[1..-1]
end
end

Now upcase is also a method of the String class, but you get my point.
